# BOC EUR/CNY 现汇卖出价爬虫（EUR/CNY Spot Rate Tracker）

这个项目用于从中国银行「外汇牌价」页面抓取 **欧元（EUR）兑人民币（CNY）的现汇卖出价**，将结果 **追加保存到同一个 `.txt` 文件**，并可从该文件 **绘制汇率变化趋势图**。

在当前版本中，项目重点是完成 **汇率数据的自动抓取、历史记录保存与趋势可视化**；后续将进一步结合新闻分析能力，扩展为一个更完整的 **换汇决策辅助工具**。

> 说明：中行外汇牌价的报价单位通常是 **“100 外币兑换人民币”**。本项目会同时保存：
> - `spot_sell_100`：100 EUR 对应的 CNY
> - `spot_sell_1`：1 EUR 对应的 CNY（由 `spot_sell_100 / 100` 换算）
>
> 数据来源：https://www.boc.cn/sourcedb/whpj/

## 项目目标 / Deliverables

### 当前已实现

- 抓取中国银行 EUR/CNY **现汇卖出价**
- 将每日数据追加保存到本地 `txt` 文件
- 读取历史数据并绘制汇率趋势图
- 支持作为日常记录工具，帮助用户持续跟踪欧元兑人民币价格变化

### 最终预期

通过每天实时分析可能影响欧元兑人民币的全球新闻，预测未来汇率走势，帮助留学生掌握换汇的最佳时机。

也就是说，这个项目的长期目标不只是“记录今天的牌价”，而是逐步发展为一个：

- **汇率数据采集器**：持续积累官方牌价历史数据
- **趋势观察工具**：通过可视化帮助用户直观理解变化趋势
- **新闻驱动的辅助预测系统**：结合全球新闻事件，对未来 EUR/CNY 走势提供参考
- **留学生换汇辅助工具**：帮助需要欧元/人民币兑换的用户更好判断换汇窗口

---

## 功能

- 抓取：欧元 **现汇卖出价**
- 存储：追加写入 `txt`（Tab 分隔，便于读取）
- 去重（可选）：同一个中行发布日（`boc_date`）默认不重复写入
- 可视化：读取 txt 并绘制趋势图（保存 png + 弹窗显示）

---

## 使用方法

### 1) 每日抓取一次并写入 txt（默认同日不重复写）

```bash
python boc_eur_sell_spot.py fetch --file boc_eur_spot.txt
```

### 2) 强制写入（即使同日已记录）

```bash
python boc_eur_sell_daily.py fetch --file boc_eur_spot_sell.txt --force
```

### 3) 绘制趋势图

```bash
python boc_eur_cny_spot.py plot --file boc_eur_cny_spot.txt --out trend.png
```
